## 🎯 Design Philosophy

### Inspiration
The new design follows principles from:
- **Stripe:** Minimal, professional, high contrast
- **Apple:** Clean typography, ample spacing, refined details
- **Notion:** Subtle interactions, glassmorphism, modern

### Key Principles
1. **Minimal over flashy:** Remove decorative elements
2. **Typography first:** Let text breathe, use hierarchy
3. **Subtle interactions:** Gentle hover effects, smooth transitions
4. **Consistent spacing:** Equal gaps, aligned elements
5. **Professional palette:** White, black, single accent color
6. **Glassmorphism:** Used sparingly for depth, not decoration
